## Manual vs automated

| Manual Testing              | Automated Testing        |
| --------------------------- | ------------------------ |
| Needs a human               | Authomated               |
| Takes time                  | Less Time                |
| Repeat for each new feature | Refactor with confidence |
| Doesn't help with code      | Write a better code      |
